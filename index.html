TimeKeeper

A fullstack web application built using Ruby on Rails, PostgresSQL, React.js, Redux, Webpack, and D3.js. TimeKeeper is a time tracking app that provides users with data visualization of their time management.

Youtube Desktop

A desktop client version of Youtube built using Electron, React.js, Redux, Webpack, and Youtube API. Youtube Desktop is a collaborative project currently in development with 5 other software engineers. The codebase was modularized and folder structure was set up based on features for easier scalability. OmniAuth 2.0 was implemented to allow access to the logged-in user's Youtube data. Also Webpack2 was utilized to split the bundle file into smaller chunks and dynamically load them as needed.

Dr. Appointment

A mobile app built using React Native, Ruby on Rails, and Twilio API. Dr. Appointment allows users to search for doctors, their availabilities, and make an appointment. Highlighted features include two-factor authentication, reminder text message using Twilio API, location-biased search using geolocation services, and auto-login using AsyncStorage.

PokeColors

A JavaScript web application built using HTML5 / CSS3, Canvas API, and jQuery. PokeColors provides users with hex-codes and color percentage of the 151 original Pokemon. It's perfect for designing and being creative with color choices. Highlighted features include analyzing SVG files pixel by pixel utilizing the Canvas API, converting the data into hex-codes, calculating the percentages, and representing them as bar graphs. Most importantly, everyone loves Pokemon!

BlindTable

A mobile app built using React Native, Ruby on Rails, Redux, Google Maps API, Twilio API, and OpenTable API. BlindTable was built for Reactathon2017 with 4 other software engineers in 2 days with goals of making our cities feel more like neighborhoods. It allows users to make reservations at nearby restaurants to meet with strangers. Highlighted features include user authentication using Twilio API, searching for restaurants and making reservations using OpenTable API, and a map view using Google Maps API.

Piano Piano Revolution

A JavaScript game built using vanilla JavaScript, HTML5 / CSS3, Canvas API, anime.js, and jQuery. Piano Piano Revolution was inspired by Dance Dance Revolution. Highlighted features include utilizing animation frame for smooth gaming experience and beautiful fireworks effects with ripple animations using anime.js.
